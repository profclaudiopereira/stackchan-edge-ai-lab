# Visão Geral

A plataforma OpenAI fornecerá:

- IA conversacional
- speech-to-text
- text-to-speech
- interação em streaming
- conversação multilíngue

Este projeto utilizará a OpenAI como principal plataforma de inteligência conversacional.

---

# Objetivos Principais

Objetivos iniciais:

- criar ambiente de APIs
- testar streaming conversacional
- avaliar qualidade PT-BR
- avaliar qualidade em inglês
- testar conversas técnicas
