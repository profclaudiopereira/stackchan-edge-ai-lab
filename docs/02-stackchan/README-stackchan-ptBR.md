# Visão Geral

O StackChan é um robô conversacional open-source baseado em ESP32-S3.

A plataforma combina:

- display
- microfones
- speaker
- servos
- Wi‑Fi/Bluetooth
- interação conversacional

Este projeto utilizará o StackChan principalmente como:

- terminal conversacional
- interface de voz
- dispositivo de interação com IA
- plataforma experimental embarcada

---

# Objetivos Principais

O StackChan será avaliado em múltiplas fases.

Objetivos iniciais:

- avaliar firmware original
- testar interação em PT-BR
- testar interação em inglês
- medir latência
- avaliar qualidade de áudio
- estudar possibilidades de integração

---

# Fase 1 — Experiência Original do Fabricante
