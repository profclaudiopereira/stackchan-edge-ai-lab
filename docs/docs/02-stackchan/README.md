# Overview

StackChan is an open-source conversational desktop robot based on ESP32-S3.

The platform combines:

- display
- microphones
- speaker
- servos
- Wi-Fi/Bluetooth
- conversational interaction

This project will use StackChan primarily as:

- conversational terminal
- voice interface
- AI interaction device
- embedded experimentation platform

---

# Main Goals

The StackChan will be evaluated in multiple phases.

Initial objectives:

- evaluate original firmware
- test PT-BR interaction
- test English interaction
- measure latency
- evaluate audio quality
- study integration possibilities

# Initial Hardware Notes

# Phase 1 — Original Manufacturer Experience

IMPORTANT:

The first tests will use the original firmware exactly as delivered by the manufacturer.

The objective is to:

- understand the original platform
- measure limitations
- document strengths
- create technical baseline
