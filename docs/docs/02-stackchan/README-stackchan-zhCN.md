# 项目概览

StackChan 是一个基于 ESP32-S3 的开源对话机器人平台。

平台集成了：

- 显示屏
- 麦克风
- 扬声器
- 舵机
- Wi‑Fi/Bluetooth
- 对话交互功能

本项目将 StackChan 主要用于：

- 对话终端
- 语音接口
- AI 交互设备
- 嵌入式实验平台

---

# 主要目标

StackChan 将分多个阶段进行评估。

初始目标：

- 评估原厂固件
- 测试葡萄牙语交互
- 测试英语交互
- 测量延迟
- 评估音频质量
- 研究集成可能性
