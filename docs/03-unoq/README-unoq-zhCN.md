# 项目概览

Arduino UNO Q 是一个混合式嵌入式平台，集成了：

- Linux 系统
- Qualcomm 处理器
- 多媒体支持
- AI 加速能力
- Wi‑Fi/Bluetooth
- 实时 MCU 架构

本项目将 UNO Q 用作：

- AI 后端
- OpenAI 网关
- 对话服务器
- 编排平台

---

初始目标：
