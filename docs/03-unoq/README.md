# Overview

The Arduino UNO Q is a hybrid embedded platform combining:

- Linux system
- Qualcomm processing
- multimedia support
- embedded AI acceleration
- Wi-Fi/Bluetooth
- real-time MCU architecture

This project will use the UNO Q as:

- AI backend
- OpenAI gateway
- conversational server
- orchestration platform

---
