# Visão Geral

O Arduino UNO Q é uma plataforma embarcada híbrida combinando:

- sistema Linux
- processamento Qualcomm
- suporte multimídia
- aceleração embarcada de IA
- Wi‑Fi/Bluetooth
- arquitetura MCU em tempo real

Este projeto utilizará o UNO Q como:

- backend de IA
- gateway OpenAI
- servidor conversacional
- plataforma de orquestração

---

Objetivos iniciais:
